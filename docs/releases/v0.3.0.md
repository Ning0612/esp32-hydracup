# HydraCup v0.3.0

## 主要更新：WebUI 風格重設計

- Dashboard、History、Settings 與登入頁統一為 HydraCup 的深色／淺色主題、導覽列、卡片、狀態橫幅與指標元件。
- History 新增月份總覽、深度分析、時段分布、週別飲水量、年度熱力圖、每日統計與最近記錄。
- Dashboard 與 History 圖表、熱力圖及卡片版面支援不同螢幕寬度；主題切換標籤統一為 `LIGHT`／`DARK`。
- Settings 重新整理提醒、蜂鳴器、感測器、Wi-Fi／Discord、NTP、MQTT 與秤重校正流程。
- README 新增 WebUI 截圖畫廊、實機照片，並更新飲水偵測 Demo 的 MP4 與 GIF。

## 安全與通知

- Normal Mode 與 ConfigPortal 加入登入 session、CSRF token、idle／absolute timeout 與登入失敗 rate limiting。
- Discord Webhook 使用憑證鏈驗證，並調整傳送失敗處理，避免在無法確認結果時重送造成重複通知。

## 修正

- 修正 annual heatmap 在窄螢幕上的響應式版面。
- 避免 Discord 每日摘要在重複觸發時重複發送。

## 驗證

- `pio test -e native` — 16/16 測試通過。
- `pio run -e esp32dev` — firmware build 成功。
- 已更新 README 版本徽章、韌體 `APP_VERSION` 與 release tag 至 `v0.3.0`。
