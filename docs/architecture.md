### иЁӯиЁҲзҙ„жқҹ

- `main.cpp` еҸӘеҢ…еҗ« `setup()` иҲҮ `loop()`пјҢ**дёҚеҗ«д»»дҪ•жҘӯеӢҷйӮҸијҜ**
- `loop()` дҫқеәҸе‘јеҸ«еҗ„жЁЎзө„зҡ„ `update()` жҲ– `loop()`
- зө•е°ҚзҰҒжӯўеңЁ `loop()` жҲ–д»»дҪ• `update()` дёӯдҪҝз”Ё `delay()`
- жүҖжңүиЁҲжҷӮдҪҝз”Ё `millis()`
- Discord HTTP POST дҪҝз”Ё FreeRTOS зҚЁз«Ӣд»»еӢҷпјҲйқһйҳ»еЎһпјү

### й—ңйҚөж©ҹеҲ¶

**`_cupLifted` ж——жЁҷ**пјҡеҸӘжңүз•¶зӢҖж…ӢзӮә `CUP_STABLE`пјҲжҲ– `POSSIBLE_DRINK`пјүдё”йҮҚйҮҸдҪҺж–ј `cupPresentThresholdGram` жҷӮжүҚиЁӯзҪ®гҖӮзўәдҝқ**йЈІж°ҙиҲҮиЈңж°ҙдәӢд»¶еҸӘиғҪйҖҸйҒҺгҖҢжӢҝиө·еҶҚж”ҫдёӢгҖҚи·Ҝеҫ‘и§ёзҷј**пјҢйҳІжӯўз§ӨеҸ°дёҠзҡ„йҮҚйҮҸеҫ®и®ҠйҖ жҲҗиӘӨеҲӨгҖӮ

**`DRINK_LIFT_TIMEOUT_MS = 120,000 ms`**пјҡиӢҘжқҜеӯҗиў«жӢҝиө·и¶…йҒҺ 120 з§’еҫҢж”ҫеӣһпјҢйҮҚзҪ® `_cupLifted` зӮә falseпјҢдёҚи§ёзҷјйЈІж°ҙдәӢд»¶пјҲиҰ–зӮәиҲҮйЈІж°ҙз„Ўй—ңзҡ„ж“ҚдҪңпјүгҖӮ

**жқҜеӯҗж”ҫзҪ®дҪҶйҮҚйҮҸи®ҠеҢ–пјҲз„ЎжӢҝиө·пјү**пјҡеҸӘжӣҙж–° `_prevStableWeight`пјҢдёҚи§ёзҷјд»»дҪ•йЈІж°ҙжҲ–иЈңж°ҙдәӢд»¶гҖӮ
